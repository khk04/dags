# Juxtagene Dags

이 프로젝트는 Juxtagene의 Dags를 관리합니다.

## 프로젝트 구조

## 요구 사항

- Python 3.x
- Apache Airflow

## 설치 방법

1. 저장소를 클론합니다.
    ```bash
    git clone https://github.com/your-repo/juxtagene-dags.git
    ```
2. 필요한 패키지를 설치합니다.
    ```bash
    pip install -r requirements.txt
    ```

## 사용 방법

Airflow 웹 UI를 통해 Dags를 관리하고 모니터링할 수 있습니다.
